# Public PM Operating System

This repo contains the public-facing content package for a commercially focused PM operating system.

It is designed to be integrated into an existing website and includes:

- a landing page spec and launch-ready copy
- a framework index page
- three core playbooks:
  - strategy
  - prioritization
  - metrics
- three attached operator playbooks:
  - strategy operator playbook
  - prioritization operator playbook
  - metrics operator playbook
- a visual system and page blueprint package for implementing the live site
- a validation checklist for reviewing tone, usefulness, and cohesion

## Audience

This content is for product managers who are actively contributing or building, not just managing process.

## Positioning

The point of view is practical, sharp, and commercially focused:

- make better product bets
- move from theory to action quickly
- use clear decision rules instead of vague best practices

This package assumes the live website exists in a separate repository. The Markdown here is written to be easy to adapt into CMS entries, MDX pages, or static site content.
